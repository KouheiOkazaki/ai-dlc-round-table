# INT-001 UNIT-003: 貸出状況ダッシュボード + 履歴

## 目的（Purpose）
全備品の現在の貸出状況をリアルタイムに可視化し、返却期限超過アラートと過去の貸出・返却履歴を提供する。

## 境界（Boundaries）
### 含むもの
- 貸出状況ダッシュボード（`/dashboard`）
  - 全備品の貸出状況一覧
  - 返却期限超過のハイライト表示
- 貸出・返却履歴ページ（`/history`）
  - 全履歴の新しい順表示
  - 備品名でのフィルタリング

### 含まないもの
- 備品マスタ管理（UNIT-001）
- QRスキャン・貸出返却操作（UNIT-002）

## 依存関係（Dependencies）
- UNIT-001: `items` テーブル
- UNIT-002: `loans` テーブル

- US-008: 返却期限超過アラートの確認
- US-009: 貸出・返却履歴の表示
