# INT-001 Intent

## 目的（Purpose）
スマートフォンでQRコードを読み取ることで、備品の貸出・返却を手軽に記録・管理できるWebアプリケーションを構築する。
管理者が備品マスタを登録・管理し、利用者がQRコードをスキャンするだけで貸出・返却が完結する仕組みを提供する。

## 成功基準（Success Criteria）
- 備品にQRコードを発行し、スマートフォンのカメラで読み取って貸出・返却を記録できること
- 現在の貸出状況（誰がいつ借りているか）をリアルタイムで確認できること
- 返却期限を設定でき、期限超過の備品をアラート表示できること
- 貸出・返却履歴を一覧で参照できること

## スコープ（Scope）
### 対象（In Scope）
- 備品マスタの登録・編集・一覧表示
- 各備品へのQRコード発行・表示
- QRコードスキャンによる貸出・返却記録
- 貸出状況のダッシュボード表示
- 貸出・返却の履歴一覧
- 返却期限超過アラートの表示

### 対象外（Out of Scope）
- ユーザー認証・権限管理（誰でも操作可能なPoC）
- メール・プッシュ通知によるアラート送信
- 備品カテゴリ・タグによる分類管理
- 複数拠点・倉庫の管理
- 外部システムとの連携（在庫管理システム等）

## 前提条件（Assumptions）
- 既存の Next.js (App Router) + Drizzle ORM + PostgreSQL 環境を流用する
- PoCとして非機能要件（性能・セキュリティ）の考慮は最小限とする
- スマートフォンのブラウザからアクセスし、カメラAPIでQRコードを読み取る

## 関連情報
- 既存コードベース: コメント投稿・一覧表示機能（`app/page.tsx`, `src/schema.ts`）
- フロントエンド: Next.js App Router + shadcn/ui
- DB: Drizzle ORM + PostgreSQL (Netlify DB)
